Code search
===========

On Software Factory, you can activate a source code search engine component, the
following component is deployed:

* `Hound <https://github.com/etsy/hound>`_: an extremely fast source code search engine

.. image:: imgs/hound.png
   :scale: 50 %

As operator, you can understand how to activate hound following the :ref:`operator
documentation <hound-operator>`.
