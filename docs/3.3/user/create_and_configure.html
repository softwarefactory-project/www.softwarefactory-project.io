Create and configure a new projectÂ¶

Creating a new project or repository is done by submitting a change on the Config repository. This repository is a regular project, see the next chapter about how to contribute to a project.

To create and/or configure a project, you need to create a new review on the config repository. Please see this example.

Once the change is approved and merged, the project will be created/updated and added to the Zuul tenant configuration.
