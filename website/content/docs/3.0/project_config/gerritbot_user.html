Gerritbot notification channels configurationÂ¶

Once the service is running (see operator configuration), you can configure the irc channels to send notifications to:

  • clone the config repository with git
  • add a new file or edit one in the config/gerritbot directory:
irc-channel-name:
  events:
    - change-created
    - change-merged
  projects:
    - myproject
  branches:
    - master
  • submit and merge the config change.
  • the gerritbot configuration will be updated with the automated post-merge run of config-update.
